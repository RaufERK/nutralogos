Nutralogos — интеллектуальный ассистент по нутрициологии (RAG)

Nutralogos — чат‑ассистент, отвечающий на вопросы о здоровье, питании и нутрициологии на основе собственной базы знаний. Проект сочетает LLM‑генерацию с поиском по векторной базе (RAG), поддерживает загрузку документов, их парсинг и индексацию, а также админ‑панель для управления настройками и синхронизацией. Формат готов к использованию как MVP: быстрые ответы, прозрачные источники, удобная загрузка знаний и стабильность под нагрузкой за счёт Redis‑лимитов и стриминга.

Технологии
- Next.js (App Router), React, TypeScript; UI: Tailwind + shadcn/ui
- Серверные маршруты и WebSocket‑стриминг ответа
- LLM: OpenAI GPT‑4o; LangChain для RAG‑пайплайна
- Векторная БД: Qdrant
- База метаданных: SQLite (better‑sqlite3)
- Rate limiting и троттлинг: Redis (локально — Docker, прод — системный сервис)
- Парсинг файлов: PDF (pdf‑text‑extract + fallback pdf‑parse), DOCX (mammoth), DOC (word‑extractor), TXT

Ключевые возможности
- Загрузка файлов и двухступенчатая дедупликация: по хешу файла (SHA‑256) и по тексту (txt_hash)
- Автоматическое извлечение текста, разбиение на чанки, эмбеддинги и индексирование в Qdrant
- Обогащение метаданных через LLM (стратегии auto/sampled/hierarchical)
- Контекст диалога; настройка «Показывать источники ответа» (кнопка всегда видна, раскрытие по настройке)
- Стриминг ответа в реальном времени по WebSocket
- Админ‑панель: настройки, статистика, синхронизация, логин

Деплой и среда
- Разделение ENV для локалки и продакшна; `REDIS_URL` отличается по окружениям
- Локально Redis в Docker, на VPS — системный Redis; для favicon применён cache‑busting
